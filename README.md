# DeFi Platform

## Prerequisites
- Node.js 18+
- npm or yarn
- Vercel CLI (optional)

## Local Development
1. Clone the repository
2. Install dependencies
   ```bash
   npm install
   ```
3. Create `.env` file with required keys
4. Run development server
   ```bash
   npm run dev
   ```

## Deployment Steps
1. Install Vercel CLI
   ```bash
   npm i -g vercel
   ```

2. Login to Vercel
   ```bash
   vercel login
   ```

3. Deploy
   ```bash
   vercel
   ```

## Environment Variables
- `VITE_ALCHEMY_API_KEY`: Alchemy Web3 API Key
- `VITE_WALLET_CONNECT_PROJECT_ID`: WalletConnect Project ID

## Tech Stack
- React
- Vite
- TypeScript
- Tailwind CSS
- Web3 Libraries
- Vercel Hosting
