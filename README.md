# DeFi Platform

## Overview
Decentralized Finance Platform with Web3 Integration

## Features
- Token Swap
- Lending
- Portfolio Tracking
- Web3 Wallet Connection

## Tech Stack
- React
- Vite
- TypeScript
- Tailwind CSS
- Web3 Libraries

## Local Development
1. Clone Repository
2. Install Dependencies
   ```bash
   pnpm install
   ```
3. Run Development Server
   ```bash
   pnpm dev
   ```

## Deployment
Deployed on Vercel with automatic GitHub integration

## Environment Setup
Create `.env` file with:
- Alchemy API Key
- WalletConnect Project ID
